# BFOR APP: Discussions Feature — App Demo

## What it does
- Displays posts pulled from a Supabase database
- Click on a post to open the full post detail page
- Scroll animations on the feed using GSAP

## Technologies Used
- HTML, CSS, JavaScript
- Supabase (database)
- GSAP / ScrollTrigger (animations)

## How to run it
1. Open the project folder
2. Run with a local server (e.g. Live Server in VS Code)
3. Make sure config.js has your Supabase URL and key

## Notes
- Posts are managed directly in the Supabase dashboard
- This is a read-only demo, users cannot submit posts
